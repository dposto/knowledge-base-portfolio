# Technical Writing Portfolio

Hi, I'm David Posto, a technical writer with experience creating knowledge base articles, SOPs, troubleshooting guides, and end-user support documentation for IT service desks and healthcare workflows.

This site is a collection of writing samples that demonstrate my documentation style: clear step-by-step procedures, consistent structure, and plain language designed for front-line support staff.

## What You'll Find Here

This portfolio contains sample documentation demonstrating:

- Knowledge base article development
- Step-by-step troubleshooting procedures
- Process and workflow documentation
- Decision-tree based support guides
- Documentation written for front-line support teams

## About These Articles

These articles are based on real help desk workflows I've documented professionally. Names, locations, and organization-specific details have been generalized to protect confidentiality.

Each article follows a structured format:

- **Symptoms** — what the caller or user is experiencing
- **Solution** — step-by-step procedures with decision branches
- **Escalation** — what to do when standard steps don't resolve the issue
